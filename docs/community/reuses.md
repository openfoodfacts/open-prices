# Reuses

## I want to reuse Open Prices data

* see [Data](../guides/data.md) for more details about the data and how to access it
* Make sure you comply with the OdBL licence
* Please get in touch at reuse@openfoodfacts.org. We are very interested in learning what the Open Prices data is used for. It is not mandatory, but we would very much appreciate it if you tell us about your re-uses (https://forms.gle/hwaeqBfs8ywwhbTg8) so that we can share them with the Open Food Facts community.
