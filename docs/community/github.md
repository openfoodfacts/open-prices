# Github

Our code is open source and available on Github.

* the backend/API code is available in [openfoodfacts/open-prices](https://github.com/openfoodfacts/open-prices)
* the frontend code is available in [openfoodfacts/open-prices-frontend](https://github.com/openfoodfacts/open-prices-frontend)
* we use `Issues` & `Discussions` for project management and technical discussions
    * [What can I work on ?](https://github.com/openfoodfacts/open-prices/issues/39)
    * [Top issues](https://github.com/openfoodfacts/open-prices/issues/229)
    * [Roadmap](https://github.com/openfoodfacts/open-prices/issues/541)
* please open a `Pull Request` related to an issue, or open a new issue if you want to contribute but don't know where to start :)
